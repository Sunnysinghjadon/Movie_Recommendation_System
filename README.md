# 🎬 Movie Recommendation System

This project is a **Movie Recommendation System** that helps users find movies based on their preferences. Leveraging **TF-IDF Vectorization** and **Cosine Similarity**, this system offers personalized movie suggestions, providing a user-friendly and intuitive experience.

## 🌟 Project Overview

The Movie Recommendation System recommends movies to users based on their chosen movie or preferences. By analyzing the content of movies, such as genres, plot summaries, and tags, the system identifies similar films and recommends them in real-time. The entire system is deployed on **Streamlit** for easy and interactive usage.

## 🛠️ Features

- **TF-IDF Vectorizer**: Converts movie descriptions into numerical data, assigning importance to each word based on its frequency in the dataset.
- **Cosine Similarity**: Measures the similarity between movies by comparing the vectors generated by TF-IDF.
- **Exploratory Data Analysis (EDA)**: Gain insights into the movie dataset, including the distribution of genres, popular movies, and more.
- **Interactive Web Application**: The system is deployed on **Streamlit**, providing a seamless user interface for movie recommendations.

## 🔍 Exploratory Data Analysis (EDA)

Before building the recommendation engine, a thorough **EDA** was conducted on the dataset, covering:
- **Popular genres**: Visualized the distribution of movies across genres.
- **Movie trends**: Analyzed the release year, ratings, and viewer preferences.
- **Word clouds**: Generated word clouds to highlight key themes in movie descriptions.

## ⚙️ How It Works

1. **TF-IDF Vectorization**: 
   - We convert the plot summaries of movies into a numerical matrix using **TF-IDF Vectorizer**. This helps to capture the relative importance of words within each movie's description.
   
2. **Cosine Similarity**:
   - Using **Cosine Similarity**, we calculate how similar each movie is to the others by measuring the angles between the vectors generated by the TF-IDF matrix.
   
3. **Recommendation Generation**:
   - For a given input movie, the system returns a list of movies that are most similar based on their plot and metadata.

## 🚀 Deployment

The Movie Recommendation System is deployed using **Streamlit**, providing a sleek and responsive interface where users can:

- Search for a movie by title.
- View a list of similar movies instantly.
- Explore other functionalities like filtering based on genres or years.

## 🛠️ Tech Stack

- **Python**: Core programming language
- **Pandas & Numpy**: Data manipulation and numerical computation
- **Scikit-learn**: Machine learning library for **TF-IDF Vectorizer** and **Cosine Similarity**
- **Streamlit**: Web framework for deploying the recommendation system
- **Matplotlib & Seaborn**: For visualizing insights during EDA

## 🎯 Key Highlights

- **Cosine Similarity**: Effectively measures the closeness of movies based on their plot descriptions and metadata.
